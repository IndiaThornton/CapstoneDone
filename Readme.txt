My Sneaker Planner 

My sneaker Planner is a web application that you can use to save information about your snaker collection. Users can save the sneakers that are currently in the collection. Recently ordered sneakers can be saved in the "em Route" list. The is a list for all the sneakers you have that the user can edit as well. Users can also add a sneaker that will be displayed to the home page of the app.
